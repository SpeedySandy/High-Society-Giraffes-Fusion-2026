# Porsche Cayenne Camper Build App (MVP)

Interactive single-page app to prototype a Porsche Cayenne camper conversion.

## Features
- Live configuration controls (model year, travel style, sleeping capacity, electrical tier, water storage).
- Dynamic budget estimate and complexity/timeline scoring.
- Auto-generated build priorities based on selected options.
- Discovery section outlining what information is needed to evolve this into a feature-rich planning app.

## Run locally
Open `index.html` directly in a browser, or serve with:

```bash
python3 -m http.server 4173
```
