# IHA AI Roadshow Quiz

Interaktives, deutschsprachiges Online-Quiz zum Thema Künstliche Intelligenz für die IHA AI Roadshow.

## Inhalt
- 10 Multiple-Choice-Fragen zu KI-Grundlagen, Prompting, Datenschutz und Governance
- Sofort-Feedback mit Erklärung nach jeder Frage
- Automatische Auswertung mit Punktzahl und Level
- Responsives Layout für Desktop und Mobile

## Start
Öffne `index.html` direkt im Browser.
