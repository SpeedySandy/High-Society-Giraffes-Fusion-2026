# Porsche Cayenne Camper Build Studio

A richer interactive web app for planning a Porsche Cayenne camper conversion.

## What this app now does
- Configure generation, model year, travel style, climate, sleeper count, water storage, power tier, and modular add-ons.
- Calculate live budget, payload impact, water autonomy, complexity score, and timeline.
- Generate a parts table and phased build plan.
- Save/load build state in localStorage.
- Export build configuration as JSON.
- Apply one-click Weekend / Expedition presets.

## Run locally
```bash
python3 -m http.server 4173
```
Then open `http://localhost:4173`.
